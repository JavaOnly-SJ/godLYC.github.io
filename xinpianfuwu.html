    Affymetrix miRNA 4.0 芯片服务

    miRNA 4.0芯片（GeneChip® miRNA 4.0 Array）是一个强大的工具，有助于研究非编码RNA的作用及其在发育和生理机制中的

    参与程度，如：蛋白翻译的抑制、可变剪切的调控、核糖体RNA的加工、mRNA的降解等。

    技术特点

    全面的覆盖：精心设计可检测miRBase Release 20数据库中所有成熟的miRNA序列

    简化的分析：可在同一张芯片上分析人类、小鼠、大鼠等所有物种的每一条miRNA

    丰富的信息：提供宿主基因ID、miRNA靶基因和miRNA聚类信息

    低的样品量：仅需低至130 ng总RNA

    服务内容

    1. 样本抽提及质检：对客户提供样本进行RNA抽提，并利用NanoDrop 及Agilent Bioanalyzer 2100进行样本质检；

    2. 芯片杂交及扫描：包括样本放大、标记、芯片杂交、洗涤、扫描及图像采集；

    3. 芯片数据分析：包括数据的均一化、差异miRNA筛选、差异miRNA靶基因预测及靶基因基因功能分析等标准分析内容；

    4. 报告整理：提供包括实验操作流程、原始数据、分析数据在内的详细实验报告。

    Affymetrix miRNA 3.0 芯片服务

    miRNA是一类21-25nt的单链小分子RNA，在基因表达、个体发育、细胞增殖与分化等生理过程中发挥着重要作用。研究显示，

    miRNA至少调控着约30%的人类基因。针对miRNA研究不断升温的情况，Affymetrix升级旗下miRNA芯片产品，推出了miRNA

    3.0 芯片。

    产品特点

    全面覆盖Sanger miRBase V17.0数据库，同时检测153个物种的miRNA

    同时检测成熟miRNA和miRNA前体（仅限人、小鼠、大鼠）

    能检测人snoRNA和scaRNA

    采用Affymetrix最新研制的FlashTag Biotin HSR RNA Labeling试剂盒，45分钟完成标记，无需扩增、纯化RNA

    服务内容

    1. 每个探针组平均包含9条探针；

    2. 参考数据库为Sanger miRBase V17.0、snoRNABase和Ensembl Archive；

    3. 高灵敏度：miRNA浓度大于1.0amol时，检出率达94% ；

    4. 高重复性：技术重复相关系数达95% ；

    5. 低样品需求量：最低仅需130ng总RNA ；

    6. 检测动态范围为3个log 。
